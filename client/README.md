# Smart Waste Management System Frontend
### <i>!!This app created using ReactJS and HERE Routing API</i>

#

## Features:
- Dashboard page
- Login page
- Responsive web page
- CRUD admin user
- CRUD employee user
- CRUD Trash
- Auto map routing based trash --> Using HERE Routing API
